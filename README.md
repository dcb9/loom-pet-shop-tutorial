Loomx Pet Shop Tutorial
========================================

## Requirements

- NodeJS
- Yarn

## Clone and install

```shell
$ git clone https://github.com/dcb9/loom-pet-shop-tutorial.git
$ cd loom-pet-shop-tutorial
$ yarn
$ yarn loom:download && yarn loom:init
```

## Run loom

```shell
$ yarn loom:run
```

## Migrate

```shell
$ yarn migrate
```

## Start

```shell
$ yarn start
```
